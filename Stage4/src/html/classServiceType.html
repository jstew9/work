Detailed Description

Class storing information about a service type.

A service type is a broad area under which sevices are classifed, ie Fire. The information stored is the name of the service area, the id of the service area, and the list of measures included in that service type. For example under the Fire service area, one measure is "Residential Fire Related Injuries per 100,000 Population (Entire Municipality)" and that measure will be stored with all the other Fire measures in a ServiceType object.
